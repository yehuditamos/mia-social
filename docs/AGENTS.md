# Agents — Mia Brain Workers

> מיה לא בנויה סביב פלטפורמות. היא בנויה סביב יכולות.
> כל Worker הוא יחידה עצמאית שמיה מפעילה לפי הצורך.

---

## סקירה כללית

```
Mia Brain (orchestrator)
│
├── 👀 Vision Worker
├── ✍️  Copy Worker
├── 🎬 Media Worker
├── 📤 Publish Worker
├── 💬 Community Worker
├── 📈 Analytics Worker
├── 📅 Planner Worker
├── ♿ Accessibility Worker
├── 🧠 Memory Worker
├── 🎨 Creative Director Worker
└── 🚀 Campaign Manager Worker
```

---

## 👀 Vision Worker

**תפקיד:** מבין תמונות וסרטונים.

**קלט:** קובץ מדיה (image / video)
**פלט:** תיאור מובנה — נושא, אווירה, אנשים, טקסט בתמונה, צבעים, קומפוזיציה

**מפעיל את:**
- Copy Worker (מקבל את התיאור כהקשר לכתיבה)
- Accessibility Worker (מקבל את התיאור ליצירת Alt text)

**דוגמה:**
```
קלט:  תמונת שיעור יוגה בחוץ
פלט:  { subject: "yoga class", mood: "calm, natural", people: 6,
         setting: "outdoor", colors: ["green", "white"], text_in_image: null }
```

---

## ✍️ Copy Worker

**תפקיד:** כותב כל תוכן טקסטואלי.

**קלט:** הקשר מ-Vision Worker + פרופיל עסק מ-Memory Worker + הנחיות משתמשת
**פלט:** כיתוב מוכן לפרסום (+ האשטאגים אם הוגדר)

**יוצר:**
- כיתובים לפוסטים
- כותרות לסטוריז
- תגובות לתגובות
- טקסט לפרסומות

**דוגמה:**
```
קלט:  { mood: "calm", business: "סטודיו יוגה נשי", tone: "חם ואישי" }
פלט:  "נשמה שקטה, גוף חזק. 🌿 מחכות לך בשיעור הבוקר..."
```

---

## 🎬 Media Worker

**תפקיד:** מעבד מדיה לפרסום.

**פעולות:**
- הורדת מדיה מ-WhatsApp
- המרת פורמטים לפי דרישות Meta
- הוספת כתוביות לסרטונים (SRT / burn-in)
- שינוי גודל ויחס תצוגה לפי פלטפורמה (Reels, Story, Feed)

**קלט:** media_url מ-WhatsApp
**פלט:** media_url מוכן לפרסום + מטאדאטה (גודל, פורמט, משך)

---

## 📤 Publish Worker

**תפקיד:** מפרסם תוכן לפלטפורמות.

**קלט:** `{ media_url, caption, platform, account_id, scheduled_at }`
**פלט:** `{ platform_post_id, published_at, status }`

**תומך ב:**
- פרסום מיידי
- פרסום מתוזמן

**פלטפורמות נוכחיות:** Instagram, Facebook
**פלטפורמות עתידיות:** TikTok, LinkedIn

---

## 💬 Community Worker

**תפקיד:** מנהל תגובות בפוסטים.

**פעולות:**
- מאזין לתגובות חדשות דרך Meta Webhook
- מסווג לפי סנטימנט (חיובי / ניטרלי / שלילי)
- מציע תגובה מנוסחת
- שולח למשתמשת לאישור לפני שליחה

**דוגמה:**
```
תגובה נכנסת: "מתי השיעור הבא?"
הצעת מיה:   "היי! השיעורים הבאים שלנו הם... 🗓"
```

---

## 📈 Analytics Worker

**תפקיד:** קורא ומסכם נתוני ביצועים.

**מקור נתונים:** Meta Graph API (reach, impressions, engagement, saves)

**פלט:**
- דוח שבועי / חודשי
- השוואה בין פוסטים
- המלצות לשיפור

---

## 📅 Planner Worker

**תפקיד:** מנהל לוח תוכן (גאנט).

**יכולות:**
- תצוגת לוח שבועי / חודשי
- תזמון פוסטים עתידיים
- תזכורות ליצירת תוכן

**משתף פעולה עם:** Publish Worker לביצוע התזמון

---

## ♿ Accessibility Worker

**תפקיד:** נגישות לכל תוכן.

**פעולות:**
- יצירת Alt text לכל תמונה (מבוסס על Vision Worker)
- בדיקת קריאות כיתוב
- תמיכה ב-RTL

**פלט:** `alt_text` שנשמר ב-Media ומוזן לפרסום

---

## 🧠 Memory Worker

**תפקיד:** הזיכרון של מיה לגבי העסק.

**שומר:**
- פרופיל עסק (שם, תחום, קהל יעד, סגנון, ערכים)
- העדפות המשתמשת (שפה, טון, האשטאגים, שעות פרסום)
- היסטוריית פרסומים
- חשבונות מחוברים (SocialAccount)

**כל Worker שואל אותו לפני פעולה.**
הוא הבסיס שמבטיח שמיה תמיד מדברת בשפה של העסק.

---

## 🎨 Creative Director Worker

**תפקיד:** יוצר חומרים ויזואליים מקצועיים.

**אחריות:**
- יצירת תמונות לפוסטים ולמודעות
- יצירת סרטונים קצרים ו-Reels
- יצירת קרוסלות (Carousel)
- יצירת סטוריז
- בחירת סגנון גרפי שמתאים למיתוג העסק
- עבודה עם מחוללי AI חיצוניים (Midjourney, DALL·E, Runway, Kling ועוד)

**קלט:** בריף ויזואלי מ-Copy Worker + פרופיל מיתוג מ-Memory Worker
**פלט:** קובץ מדיה מוכן (image / video) + מטאדאטה של סגנון

**עיקרון:** Creative Director Worker לא קשור למחולל ספציפי. הוא מגדיר מה נדרש — המנוע הוא פרט יישומי הניתן להחלפה.

---

## 🚀 Campaign Manager Worker

**תפקיד:** בונה ומנהל קמפיינים מלאים.

**אחריות:**
- בניית קמפיין מקצה לקצה (מסר → תוכן → תזמון → פרסום)
- תכנון רצף תוכן מסודר לאורך זמן
- התאמת מסרים לפי שלב (מודעות / שיקול / המרה)
- ניהול השקות (מוצר חדש, מבצע, אירוע)
- המלצות על תזמון ותקציב

**קלט:** מטרת קמפיין + תאריכים + תקציב (אופציונלי)
**פלט:** לוח תוכן מפורט + הנחיות לכל Worker הרלוונטי

**משתף פעולה עם:** Planner Worker (תזמון), Copy Worker (מסרים), Creative Director Worker (ויזואלים), Publish Worker (ביצוע)

---

## עקרונות

| עיקרון | פירוט |
|--------|-------|
| Workers עצמאיים | כל Worker ניתן לקריאה ישירה ולבדיקה בנפרד |
| Memory Worker ראשון | כל Worker מתחיל בשאילת Memory Worker |
| פלטפורמה = יעד בלבד | Workers לא יודעים "אינסטגרם" — הם יודעים "פרסום" |
| Mia Brain מתאמת | היא היחידה שיודעת איזה Workers להפעיל ובאיזה סדר |
| מנוע AI = פרט יישומי | מיה אינה תלויה במודל AI יחיד. כל Worker משתמש במנוע המתאים ביותר למשימה שלו, כך שניתן להחליף או לשדרג מנועים בעתיד ללא שינוי במערכת כולה. |
