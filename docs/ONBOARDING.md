## שלב 3 — חיבור Meta (OAuth)

```
עכשיו נחבר את האינסטגרם והפייסבוק שלך 🔗

לחצי על הקישור הזה כדי לאשר הרשאות:
👉 [קישור OAuth של Meta]

את לא צריכה להעתיק שום דבר —
אחרי שתאשרי, אני כבר אטפל בכל השאר 💜
```

**מה קורה ברקע:**
1. המשתמשת לוחצת על הקישור → נפתח חלון ההרשאה הרשמי של Meta
2. היא מאשרת גישה לפייסבוק + אינסטגרם
3. Meta מחזיר `code` → הבקאנד מחליף ל-`access_token`
4. מיה שומרת `access_token`, `page_id`, `instagram_account_id` אוטומטית
5. מיה שולחת אישור:

```
מעולה! ✅ חיברתי בהצלחה:
• פייסבוק: [שם הדף]
• אינסטגרם: @[handle]

יעד: פחות מ-2 דקות לחיבור מלא.
```

## אישור לפני פרסום — שיחה טבעית

מיה לעולם לא תשתמש בכפתורים לאישור. האישור הוא שיחה.

```
מיה: "כך ייראה הפוסט —

[תצוגה מקדימה של הכיתוב]
[תמונה]"
```

**תגובות שמיה מבינה:**

| מה המשתמשת כותבת | מה מיה עושה |
|------------------|-------------|
| מאשרת / תעלי / יאללה / בסדר / כן | מפרסמת מיד |
| תשני את הכותרת | מבקשת כותרת חדשה |
| תוסיפי אימוג'י | מוסיפה אימוג'ים לכיתוב |
| תחליפי את התמונה הראשונה | מבקשת תמונה חדשה |
| תזמני למחר | מבקשת שעה ומזמנת |
| לא / בטלי | מבטלת ושואלת מה לשנות |

## עצירה באמצע Onboarding

עצירה היא פיצ'ר, לא כישלון.

```
יהודית,
עצרנו בשלב 4 מתוך 7.
כשתרצי להמשיך — פשוט כתבי "ממשיכים" 💜
```

**כללים:**
- כל תשובה נשמרת מיד — אין אובדן מידע
- timeout של 24 שעות → מיה שולחת תזכורת עדינה
- ניתן לעצור עם המילה "עצור" בכל שלב
- ניתן לחזור אחורה עם המילה "חזור"
